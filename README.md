ArumaRoom
=========

:boom:

Control light and music with your hands.

Installation
------------

```
> nvm use 7.6.0
> yarn install
> quasar dev
```

Audio
-----

```
> ffmpeg -i name.mp3 -strict -2 name.webm
```
